How these files were produced
- Furniture sprites: native pack PNGs upscaled 4x NEAREST (1 src px = 4 dst px).
- floor_wood / floor_tile: pack grey floor tiles recolored to the pack's own warm
  wood/beige palette.
- wall / wall_baseboard / window / rug: DERIVED by us from the pack's exact palette
  (beige 202,198,185; trim 48,42,40; wood 207,168,109/179,136,87/136,92,71). These

Pack: "Ninja Adventure - Asset Pack"

License: CC0 1.0 (public domain). Attribution not required but appreciated.

How these files were produced (by us)
- Picked 4 distinct civilian villagers for an office cast:
    boss=char9 (white-haired elder), manager=char6 (salaryman w/ tie),
    sales=char25 (casual), clerk=char17 (blonde).
- Took only the front-facing idle frame (col 0, row 0) of each 64x112 (16x16 grid) sheet.
- Recolored ONLY each outfit/shirt to its role color via HSV (kept source Value =
    shading preserved): boss red #c0392b, manager blue #2980b9, sales green #27ae60,
    clerk purple #8e44ad. Hair/skin/face untouched.
- Composited on a 16x18 canvas (1px headroom) and upscaled 4x NEAREST -> 64x72 to
    match the furniture scale. char2_<role>.png == char2_<role>_0.png (rest);
    char2_<role>_1.png is the same sprite 1px higher (=4px @4x) for a tiny breathing bob.
